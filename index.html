Aneyeon Pets (WEB Veterinarias)

ISVDR - Backend Developer 2023 - Actual

BACKEND DEVELOPER - CO-FOUNDER & LEAD DEVELOPER (ANEYEON PETS) 2022 - 2023


Para mi proyecto final de la carrera, me encuentro desempeñando un rol de co-líder en el desarrollo de una Aplicación Web destinada a clínicas veterinarias, utilizando Node.js como tecnología central. En este proyecto, mi principal responsabilidad radica en la creación y gestión de una API esencial para el manejo de datos críticos.

Además de la creación de esta API, hemos enfocado nuestros esfuerzos en fortalecer la seguridad de la aplicación mediante la implementación de cookies y JSON Web Tokens. Esto garantiza la protección adecuada de la información sensible.

Una parte destacada de nuestro proyecto es la exitosa integración con servicios de terceros, como Mercado Pago, AFIP y Google. Esta colaboración con estos proveedores externos ha enriquecido significativamente la funcionalidad y el alcance de nuestra aplicación.

Asimismo, he contribuido activamente a la implementación de la API en el FrontEnd, haciendo uso de JavaScript para garantizar una experiencia de usuario fluida y coherente. En conjunto, estos esfuerzos han permitido avanzar de manera significativa en el desarrollo de la aplicación web para veterinarias.
